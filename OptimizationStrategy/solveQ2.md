# Q2

基于品类的最优化。

## get D_i first

成等效打折率 $D_i$ ：定义 $A_{i_1}$ 与 $A_{i_2}$ 分别作为 $D_i$ 中的打折单品的指标集合与未打折单品集合，则有如下关系得到 $D_i$：
$$
D_i = \frac{ \sum_{(i,j) \in A_{i_1}} p_{i,j} / |A_{i_1}| } { \sum_{(i,j) \in A_{i_2} } p_{i,j} / |A_{i_2}| } 
$$

需要基于附件二进行处理。

附件二的表头如下：

销售日期	扫码销售时间	单品编码	销量(千克)	销售单价(元/千克)	销售类型	是否打折销售
